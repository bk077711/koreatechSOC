    네이버 지도 API

    네이버 지도 API를 이용하여 검색된 낚시터의 위치를 화면상에 표시하여준다.

    위도 경도를 기준으로 낚시터의 위치를 구하고 지도에 표시할 수 있다.

    검색된 낚시터가 아닌 인근에 있는 낚시터도 표시하여 준다.

    Web Dynamic Map
    패닝, 줌인아웃 등 인터랙티브가 가능한 동적 지도를 보여줍니다.
    파라미터 타입 세부설명
    center string 중심 좌표와 X,Y좌표를 따르고 <경도 위도> 순서
    w,h int 가로 세로 이미지 크기
    level int 지도의 줌 레벨

    전국낚시터 API

    공공데이터포털에서 제공하는 전국 낚시터 정보를 받아와서 필요한 정보를 추출한다.

    여기서 필요한 정보는 낚시터 위치, 가격, 어종 등이다.

    추출된 데이터를 DB에 저장하고 화면에 표기하기 위해 파싱한다.

    전국낚시터정보
    전국 낚시터 정보(주요 어종, 낚시터 편익시설)등을 제공합니다.
    파라미터 타입 세부설명
    fshlcNm string 해당 낚시터 이름
    rdnmadr string 낚시터의 도로명 주소
    kdfsh string 해당 낚시터에서 잡히는 어종
    usecharge int 낚시터 이용 비용
    latitude string 해당 낚시터의 위도
    longitude string 해당 낚시터의 경도

    해양 수온 API

    전국의 관측소로부터 해양 수온(심층,중층,표층)의 정보를 받아온다.

    받아온 데이터를 서해,남해,동해로 나눠서 평균값을 매긴 후 해양수온을 지도에 표시해준다.

    물고기마다 적서 수온이 다르기 때문에 해양 수온에 따라 낚시터 선택에 큰 영향을 미친다.

    실시간어장정보
    (해양수산연구정보) 전국 연안 관측소에서 측정한 수온, 염분, 용존산소 등의 정보를 실시간 매 30분 단위로 서비스
    파라미터 타입 세부설명
    STA_CDE string 관측소 코드
    GRU_NAM string 해양 이름(동해,서해,남해)
    wtrTmp string 해양 표층 수온
    SDATE string 해양 수온 관측 시작 시간

    사용 설명서

    시도군 등 가고자하는 주소를 입력합니다.

    DB에서 주소에 해당하는 낚시터를 검색합니다.

    검색된 낚시터가 있는 바다가 동/서/남해인지 구분 후에 해양수온을 계산하여 보여줍니다.

    수온에 따른 추천 어종이 있는 낚시터를 지도에 마커로 표시하여 추천해줍니다.

    파라미터 소개
    입출력 파라미터 타입 설명
    Input 지역 varchar 가려고 하는 지역의 주소
    Output fname varchar 해당 낚시터의 이름
    location varchar 낚시터의 도로명 주소
    fish varchar 해당 낚시터에서 잡히는 어종
    fmoney varchar 낚시터 이용 비용
    flatitude varchar 해당 낚시터의 위도
    flongitude varchar 해당 낚시터의 경도
    temp varchar 낚시터 인근 바다의 평균 수온
    fishlist varchar 해양 수온에 적합한 어종 추천
